# **App Name**: LabWise

## Core Features:

- Patient Management: Add new patients or modify existing patient data, including full name, date of birth, gender, and contact information.
- New Test Order Creation: Create a new test order by selecting a patient, adding one or more tests (with manually entered test prices), and selecting the lab type (In-House or Outside). Includes a manual discount input.
- Automated Billing: Calculates the bill total, applying a manual discount if present.
- Dashboard Overview: View a dashboard displaying all test orders, separated into two tabs: 'In-House' and 'Outside' tests.
- Test Result Entry: Enter test results for pending orders, including test name, result value, normal range, and technician notes.
- PDF Report Generation: Tool that generates PDF reports containing patient data and test results. Function only calculates the current manual discount.
- Data Export: Export test order data of the in-house and outside type, into CSV format. An excel-compatible sheet can then be produced.

## Style Guidelines:

- Primary color: Deep teal (#008080) to convey a sense of trustworthiness and precision.
- Background color: Light, desaturated teal (#E0F8F7) for a clean and calm interface.
- Accent color: Soft orange (#D9A663) for interactive elements and important calls to action.
- Body and headline font: 'Inter', a grotesque-style sans-serif, will be used, because it conveys a modern, objective and neutral appearance.
- Code font: 'Source Code Pro' for displaying code snippets.
- Use clear and simple icons that represent various lab tests and functions.
- Use a clean and organized layout to ensure easy navigation and readability.
